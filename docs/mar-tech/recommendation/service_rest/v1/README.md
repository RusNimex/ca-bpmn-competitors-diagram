# REST API рекомендаций 1.0.0

**Зачем:** Виджет рекомендаций конкурентов на дашбордах. Цель — больше аккаунтов → выше тариф.

**Кто:** Пользователь — владелец дашборда с аккаунтами соцсетей.

**Стратегии 1.0.0:**
1. **PBUW** — коллаборативная фильтрация («с этим часто покупают»); общий список
2. **CATG** — по категории соцсети; для дашборда выбранного аккаунта (требуются social, socialId)

**API:** `GET /api/v1/account_recommendation` — algo=PBUW | CATG, limit 1–50.
