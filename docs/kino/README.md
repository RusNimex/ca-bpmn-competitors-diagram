Проектируем приложение для покупки билетов в кинотеатр. Пользователь заходит на сайт, находит фильм и доступные кинотеатры, выбирает зал, места и покупает билет.
Задачи:

1. Собрать требования у заказчика (интервьюера)
2. Sequence-диаграмма
3. ER-диаграмма базы
4. Как будет выглядеть запрос поиска фильмов
5. Спроектировать тело ответа на этот запрос
6. Какие средства мониторинга и сбора статистики по приложению ты бы внедрил
