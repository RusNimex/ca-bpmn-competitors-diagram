# sa-docs-projects

## Примеры работ

Репозиторий содержит документацию, SRS, диаграммы и артефакты по проектам, в которых я участвовал в роли системного аналитика.

## Структура репозитория

| Проект | Описание | Артефакты |
|--------|----------|-----------|
| **mar-tech** | MarTech-продукт: рекомендации конкурентов, интеграции | SRS, бизнес-правила, глоссарий |
| **kino** | Приложение для покупки билетов в кинотеатр | SRS, ER-диаграмма, Sequence-диаграммы |
| **telecom** | Оптимизация авторизации (альтернативы SMS) | Описание задачи, контекст решения |
| **farm-market** | Процессы оформления заявки | BPMN (as-is / to-be) |

## Примечание

- **Структура** — каждый каталог в `docs/` — отдельный кейс.
- **Примеры документов** — SRS с нумерацией требований, глоссарием, табличной структурой.
- **Диаграммы** — ER, Sequence, BPMN в форматах PlantUML, Devart, Camunda.
